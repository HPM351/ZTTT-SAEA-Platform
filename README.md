# 高功率微波优化 Web 平台 (HPM_OPT_WEB)

本项目是一个专业的高功率微波优化 Web 平台。平台集成了 CST 仿真参数扫描、代理模型辅助的演化算法优化（SAEA）、神经网络预测以及大语言模型分析等核心引擎。系统整体采用全模块化、去物理化的数据契约设计，保障了极高的扩展性。

---

## 1. 运行环境准备

在开始部署之前，请确保您的本地计算机已安装以下基础运行环境：

* **后端环境**：Python 3.x 及 pip 包管理工具，建议3.9-3.12版本，较为稳定。
* **前端环境**：Node.js 与 npm 包管理工具。
* **仿真环境**：CST Studio Suite 并建议使用2024及以后的版本。
* **本人采用的环境配套为3.10.11+CST 2024，实际运行较为稳定可以借鉴**

---

## 2. 后端部署

项目后端主要基于 Python 构建，入口文件为 main.py。请按照以下步骤启动：

1. 进入项目根目录 HPM_OPT_WEB/。
2. 安装后端依赖。在终端中执行以下命令读取 requirements.txt 并安装必要库：

   pip install -r requirements.txt


---

## 3. 前端部署

项目前端代码位于 hpm-frontend/ 目录，基于行业标准的 Vite 框架进行构建。

1. 在终端中切换至前端项目目录：

   cd hpm-frontend

2. 安装前端核心依赖项。系统将读取 package.json 和 package-lock.json 进行精准安装：

   npm install

> 注意：当前有两种启动方式。一般情况下推荐第一种，直接运行start.py，终端打印出“数据库与 SAEA 引擎初始化完成”以及三个网址后即为启动成功，直接去网页登录网址即可
>
> 如果这个方法无法使用，采用方法二手动启动。先运行main.py，随后打开终端，输入cd hpm-frontend，执行成功后接着输入npm run dev

---

## 4. 基础运行与简单优化流程

完成前后端双重启动后，建议按照以下步骤执行一次基础的参数优化任务以确认功能是否落实：

1. **访问平台主页**
   通过浏览器访问前端开发服务器提供的本地地址（通常为 localhost），进入系统主页。

2. **配置优化任务**
   在系统导航栏中，根据需求选择进入 CST 参数扫描页面或 SAEA 优化页面。

3. **定义变量与目标**
   当前支持基于cst文件路径一键导入cst配置，自动弹出器件的所有参数。根据前端提示设置好所有的参数与目标后尝试启动一次仿真或优化

4. **触发并监控任务**
   点击执行按钮，前端系统将向后端发送标准 JSON 格式的 Payload 触发任务。任务开始后支持前端页面路由的任意切换甚至包括关闭网页，但是不可终止前后端进程。考虑到CST的COM接口性能限制，当前优化不允许并行优化，建议排队串行优化。
